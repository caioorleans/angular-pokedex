# App: Pokedex
## Aplicação no qual é possível consultas todos os monstrinhos de bolso da primeira geração.

### Principais conhecimentos adquiridos ou exercitados com esse projeto:
- Consumo de API REST, onde utilizou-se a API PokeApi obtenção dos dados;
- Uso dos módulos AngularSvgIcon e AngularSvgIconPreloader para fazer uso de ícones SVG;

### Link para uso da aplicação:
